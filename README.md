#Expense Tracker
##overview
A simple console-based expense tracker built with pythn.This project allows the user to add expenses(amount,category,date),view all expenses and calculate total expenditure.
##Features
- Add expense(amount,category,date)
- view all recorded expenses
- calculate total expenses
##Technologies
-python 3.13
#How to run
1.open terminal.
2.'python expense_tracker.py'
3.Follow the menu options(1-Add,2.-view,3-Total,4-Exist)
