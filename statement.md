#project statement
##project Title
Expennse Tracker
##problem statement
Managing daily expenses becomes difficult when users do not have a proper system to record transactins.small day-to-day expenses are often forgotten,which causes confusion later.This project provides a simple amd effective python-based solutin fr trackig expenses.
##Scope of the project
-The system allows users to add expenses with amount,category and date.
-users can view all recorded expenses.
-The system can calculate total expenses.
-The project is console-based and suitable for beginners learning python.
-Data is maintained only during runtime(no database used)
##Target users
-students
-Individuals who want a simple expense-tracking tool
-Anyone learning python and building beginner-level projects
##High-level Features
-Add Expense
-view all expenses
-calculate Total expenses
-Menu-driven interface
